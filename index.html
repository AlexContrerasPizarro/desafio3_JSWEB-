Desafío 3 - Funciones

Parte 1

Transforma esta declaración de función a una función de expresión. El ejercicio debe quedar en un archivo nombrado `1 funcion.js`

 
    function example(a, b, c){
    return a+b+c
    }

Solución

 
    const example = function(a, b, c) {
    console.log(a + b + c);
    }   
                        
    example(3,2,1);

Parte 2

Transforma las siguiente función a una arrow function de una línea. Este ejercicio debe quedar en un archivo llamado `2 arrow.js`

 
    suma = function(a, b){
    return a + b
    }

Solución


    const suma = (a,b) => console.log(a + b);

    suma(3,2);
